
# 整体风格

代码整体采用All-man风格，除了以下的特例

1. 代码下级只有一行，例如：
```
	if (i == 0) 
		break;
		
	for (int i = 0; i < n; ++ i)
		printf("%d", a[i]);
```

2.  如果有`if, else if, else`语句，不管几行都坚持花括号

# 符号习惯

1. 所有符号后面都要打空格
2. 使用`++ i`代替`i ++`，其余都类似

# 命名规范

1. 函数名称必须以下划线命名法进行命名
2. 局部变量用下划线命名法进行命名
3. 全局变量所有字母全部大写，用下划线命名法
4. 名称应语义清晰，直接表示该函数或者变量的作用
