# P3 需求文档

## 1. 负责模块

- `src/report.c` / `include/report.h`
- `src/ui.c` / `include/ui.h`

## 2. 目标简介

P3 负责系统的查询展示与用户交互，实现菜单界面、查询条件输入、结果显示和报表生成。

## 3. 主要功能

### 3.1 菜单与交互

- 主菜单：展示系统功能入口，如物品管理、借用、归还、查询、报表、退出。
- 子菜单：根据不同功能提供操作选项，并引导用户完成输入。
- 输入验证：对用户输入进行基本格式检查，提示非法输入。

### 3.2 查询功能

- 按物品编号查询物品详情。
- 按物品名称查询物品记录。
- 按借用人查询借用/归还记录。
- 支持显示当前库存、借用状态和归还情况。

### 3.3 报表生成

- 生成物品出入账报表，汇总借用与归还数据。
- 显示库存变更、借用统计、归还统计等信息。
- 提供报表文本输出格式，便于保存或打印。

### 3.4 错误提示与反馈

- 提示用户操作结果，如成功、失败、无数据等。
- 对关键业务步骤进行确认与提示。

## 4. 与其他模块协作

- 从 `item` / `borrow` 模块获取查询数据。
- 调用 `storage` 模块加载和保存数据。
- 作为入口层，负责调用各业务模块处理用户请求。
