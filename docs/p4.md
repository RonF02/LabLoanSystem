# P4 需求文档

## 1. 负责模块

- `src/storage.c` / `include/storage.h`
- `src/utils.c` / `include/utils.h`

## 2. 目标简介

P4 负责实现系统底层数据存储与公共工具功能，保证数据读写可靠、输入处理安全，并为上层业务提供基础支持。

## 3. 主要功能

### 3.1 数据持久化

- 读取数据：从 `data/items.txt`、`data/borrow_records.txt`、`data/return_records.txt` 加载数据。
- 保存数据：在新增、修改、借用、归还等操作后，将变更写回 TXT 文件。
- 文件格式：定义统一的数据存储格式，支持文本解析和序列化。
- 容错处理：处理文件不存在、读写失败、数据格式异常等情况。

### 3.2 文件路径与目录管理

- 定义数据文件默认路径。
- 初始化时检查并创建必要数据目录与文件。

### 3.3 通用工具函数

- 输入读取：安全读取用户输入字符串、整数、日期等。
- 字符串处理：去除换行、空格、格式化输出等。
- 日期验证：校验日期格式、比较日期顺序、判断有效性。
- 错误处理：封装常用错误提示与返回值。

## 4. 与其他模块协作

- 为 `item`、`borrow`、`report` 模块提供数据读写接口。
- 为 `ui` 模块提供输入处理工具。
- 通过公共工具降低重复代码与风险。

## 5. 交付内容

- `src/storage.c`、`include/storage.h`
- `src/utils.c`、`include/utils.h`
- 数据格式说明文档
- 测试用例与异常处理说明
