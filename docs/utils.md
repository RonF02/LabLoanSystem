# utils 模块使用说明

## 文件

- `include/utils.h`
- `src/utils.c`

## 作用

`utils` 模块提供系统的基础工具函数，主要用于安全读取用户输入、日期格式校验和字符串比较，避免重复实现这些常见功能。

## 提供的接口

### `bool read_line(char *buffer, int size)`

- 作用：从标准输入读取一行文本，并去除末尾换行符。
- 参数：
  - `buffer`：存放读取内容的目标缓冲区。
  - `size`：缓冲区大小。
- 返回值：
  - `true`：读取成功。
  - `false`：读取失败（例如遇到 EOF）。
- 使用建议：用于替代直接 `fgets`，避免保留换行符并方便后续字符串处理。

### `bool is_valid_date(const char *date)`

- 作用：检查日期字符串是否符合 `YYYY-MM-DD` 格式，并检验年月日是否构成合法日期。
- 参数：
  - `date`：待校验的日期字符串。
- 返回值：
  - `true`：合法日期。
  - `false`：格式不正确或日期无效。
- 使用建议：用于校验用户输入的借用/归还日期。

### `int safe_strcmp(const char *a, const char *b)`

- 作用：在比较字符串时避免传入 `NULL` 指针导致未定义行为。
- 参数：
  - `a`：待比较字符串。
  - `b`：待比较字符串。
- 返回值：与标准 `strcmp` 相同：
  - `< 0`：`a` 小于 `b`
  - `0`：相等
  - `> 0`：`a` 大于 `b`
- 使用建议：用于所有可能包含空指针的字符串比较场景。

## 代码实现简要说明

- `read_line` 使用 `fgets` 读取输入，并将末尾的换行符替换为字符串结束符。
- `is_valid_date` 先校验字符串长度和分隔符位置，然后判断年月日数值范围，并处理闰年情况。
- `safe_strcmp` 将 `NULL` 转换为空字符串后调用 `strcmp`。

## 注意事项

- `read_line` 假定输入缓冲区足够大，如果输入超过 `size - 1` 字符，剩余字符会保留在 stdin 中。
- `is_valid_date` 不支持时间部分，仅适用于日期格式输入。
- `safe_strcmp` 适合用于防御性编程，但仍建议在调用前尽量保证字符串不为 NULL。
