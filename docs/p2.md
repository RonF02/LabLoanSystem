# P2 需求文档

## 1. 负责模块

- `src/item.c` / `include/item.h`
- `src/borrow.c` / `include/borrow.h`

## 2. 目标简介

P2 负责实验室物品管理与借用归还业务逻辑，实现系统核心业务功能，确保物品库存、借用记录和归还记录的一致性。

## 3. 主要功能

### 3.1 物品管理

- 新增物品：录入物品编号、名称、型号、库存数量和描述。
- 修改物品：支持更新物品信息和库存数量。
- 删除物品：删除不再使用的物品信息。
- 查看物品：按编号或名称检索物品，并显示库存与基本信息。

### 3.2 借用管理

- 借用登记：录入借用人、物品编号、借用数量、借用日期和预计归还日期。
- 库存检查：借用前检查物品库存数量，库存不足时拒绝借用。
- 同步库存：借出时减少库存，可借数量动态更新。

### 3.3 归还管理

- 归还登记：录入实际归还数量和归还日期。
- 更新借用状态：标记已归还或部分归还的借用记录。
- 恢复库存：归还后按实际数量增加库存。

## 4. 与其他模块协作

- 与 `storage` 模块协作，获取当前物品列表和借用记录。
- 与 `ui` 模块协作，接收用户输入和输出业务结果。
- 与 `report` 模块协作，提供查询结果与统计基础数据。
